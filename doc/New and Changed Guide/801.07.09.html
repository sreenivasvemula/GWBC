Changes to Return Premium Plans

A List Bill account can be the payer account for policies. When there is a return premium on a fully paid policy, the excess credit was automatically put into the List Bill payer account’s Unapplied fund. The common requirement for this scenario is that the excess return premium be sent to the owner account.
Return premium plans now provide the option to send excess return premium for List Bill policies to the owner account. Return premium plans now have a new setting called List Bill Account Excess with the supported values of Policy Owner Unapplied and Policy Payer Unapplied.
