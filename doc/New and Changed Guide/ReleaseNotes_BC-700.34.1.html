Release Information

These release notes apply only to this release of Guidewire BillingCenter.

Version Number

This release of Guidewire BillingCenter is 7.0.0.
•
For Standard Reporting, BillingCenter 7.0 supports IBM Cognos Business Intelligence 10.1 as the report server.
•
If you are integrating with Guidewire PolicyCenter, this version of BillingCenter requires PolicyCenter 7.0.0 or later. If you wish to integrate with PolicyCenter 7.0.0 or 7.0.1, see “Cannot integrate with PolicyCenter 7.0.0 or 7.0.1 (BC-8554)”.
