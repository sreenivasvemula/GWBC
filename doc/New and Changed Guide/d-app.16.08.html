Agency Bill Exception Persistence

BillingCenter 7.0 adds agency bill exception persistence. BillingCenter now persists whether or not an invoice item is currently an exception and the date the exception occurred. The new IInvoiceItem plugin allows you to configure the behavior of invoice item exceptions. See “Invoice Item Plugin”. In the base configuration agency bill exception is unchanged.
