Identifying Eligible Invoice Items

During the credit handling process, BillingCenter filters invoice items to identify and select items eligible to be paid by a credit allocation.
One filter criteria is specified by the Positive Item Qualifier property contained in the return premium plan. The base configuration provides several property settings that are described in the section “Positive Item Qualifier” in the Application Guide.
The filter criteria of the Positive Item Qualifier are initialized in the DirectBillPayment plugin method addPositiveItemsDistributionCriteria. By modifying the implementation of this method, the filter criteria can be customized to meet unique requirements. For additional details, refer to “Customize the Filter Criteria for Credit Allocations” in the Integration Guide.
