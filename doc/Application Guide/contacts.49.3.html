Adding an Existing Contact to an Account

Contacts can be shared among accounts and producers. To add an existing contact to an account, perform the following steps.
1.
With the relevant account active, click Contacts to show the Contacts screen.
2.
Click Edit and then click Add Existing Contact.
3.
Use the Search Criteria to find the contact.
4.
Click the Select button next to the desired contact.
5.
If necessary, update the contact information. One or more roles can be assigned to the contact by clicking Add.
6.
To save the contact, click Finish.
