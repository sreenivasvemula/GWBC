BillingCenter Application Guide : Accounts, Policies, and Producers : Policies

Policies

This topic discusses setting up and maintaining a policy which is a basic BillingCenter entity. The first step in the billing process is setting up new accounts and their associated policies.
Much of the information needed to set up an account and its policies comes from the Policy Administration System (PAS) integrated with BillingCenter. However, in the BillingCenter base configuration, a billing clerk can use the user interface to create policies.
