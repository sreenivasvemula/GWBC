Policy Administration System Integration

The BillingCenter base configuration includes a completely functional integration with Guidewire PolicyCenter. Alternatively, you can integrate BillingCenter with another policy administration system of your choice. This topic describes how BillingCenter integrates with any policy administration system. It also provides specific information about how BillingCenter integrates with PolicyCenter.
