BillingCenter Rules Guide : Gosu Business Rules : Rule Set Categories : Rule Set Summaries

Rule Set Summaries

BillingCenter Studio contains the following sample rule sets:
 
Rule set category      
Contains rules to
Assignment
Determine the responsible party for an activity or trouble ticket
Event Message
Handle communication with integrated external systems. See the “Messaging and Events” in the Integration Guide for information on events and event messaging
Exception
Specify an action to take if an Activity is overdue and has entered the escalated state.
Preupdate
Trigger automatic actions anytime that an account or other high-level entity changes.
Validation
Validate a small subset of the BillingCenter entities.
