Billing Summary Web Service APIs (BillingSummaryAPI)

BillingCenter publishes the BillingSummaryAPI web service for external systems to get billing summaries for accounts and policies. This web service is intended primarily for InsuranceSuite integration with PolicyCenter.
Note: The billing summary methods use Gosu class DTOs without the suffix DTO.
The following table summarizes when and how to use each billing summary method.
 
To retrieve
Call this method
Description
account-level billing summary
retrieveAccountBillingSummary
Takes an account number and returns an account billing summary designed for PolicyCenter integration. The result type is an array of BCPCAccountBillingSummary DTO objects. See the DTO Gosu class in Studio for a list of properties.
policy-level billing summary for an account
retrievePeriodsBilledToAccount
Takes an account number and a term number (an int). Returns a billing summary for that policy suitable for display in the PolicyCenter user interface. An account billing summary contains information like current and past amounts due. The method return type is an array of PolicyBillingSummary DTO objects. That DTO contains money values as instances of the MonetaryAmount type.
open policy periods that the given account pays but does not own
retrievePeriodsBilledToAccount
Takes an account number and returns a displayable policy period as a DisplayablePolicyPeriod DTO object. See the DTO Gosu class in Studio for a list of properties.
open policy periods owned by the given account
retrievePeriodsForAccount
Takes an account number and returns a displayable policy period as a DisplayablePolicyPeriod DTO object. See the DTO Gosu class in Studio for a list of properties.
invoices of a specified account
retrieveInvoicesForAccount
Takes an account number and returns invoices of the specified account as an array of PCInvoiceInfo DTO objects. See the DTO Gosu class in Studio for a list of properties.
