Batch Processing Requirements

After successfully loading and importing your legacy data into BillingCenter, run the following types of batch processing in the order listed:
1.
Database Statistics
2.
Charge Pro Rata Transaction
3.
Write-off Staging
4.
Invoice
5.
Legacy Collateral
6.
New Payment
7.
Legacy Delinquency
8.
Legacy Agency Bill
9.
Statement Billed
10.
Invoice Due
11.
Statement Due
12.
Database Statistics, a second time
13.
Policy Closure
14.
Commissions Payable Calculations
15.
Producer Payment
See also
•
For information on running the types of batch processing listed above, see “List of Work Queues and Batch Processes” in the System Administration Guide.
