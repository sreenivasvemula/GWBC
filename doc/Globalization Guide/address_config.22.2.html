Addresses in BillingCenter

BillingCenter contact screens display address information for contacts. BillingCenter provides multiple PCF address modes to represent address formats that vary by country.
Read-only and Editable Addresses
BillingCenter displays address information as read-only text or as editable text entry fields:“Understanding Global Addresses”
•
BillingCenter displays read-only addresses as read-only text on multiple lines. BillingCenter uses the country.xml file for the current country and the AddressFormatter class to determine which address fields to show.
•
BillingCenter displays editable addresses as a set of editable text fields in which you can add, modify, or delete information. BillingCenter uses the country.xml file for the current country to determine the address fields to show.
BillingCenter displays a range selector—a drop-down list—in the address screen to enable you to select from an array of appropriate addresses. If you cannot edit an address, the address fields are dimmed. The drop-down list can also display a New command that you can use to create a new address.
Address Owners
BillingCenter builds the address view around the concept of an address owner. The address owner identifies the object that owns a particular address. Additionally, the address owner controls how the address widget looks in the user interface and ensures that BillingCenter saves the address properly. You can define different address owners depending on your requirements.
