Running the Commission Payable Calculations Process

Run the Commission Payable Calculations process on your starting version before you upgrade the database. The Commission Payable Calculations process makes commission payable on direct bill polices.
To run the Commission Payable Calculations process
1.
Start your pre-upgrade BillingCenter server.
2.
Open BillingCenter and log in with an administrator account.
3.
Open Server Tools by pressing ALT + SHIFT + T.
4.
Click Batch Process Info.
5.
In the Action column for Commission Payable Calculations, click Run. Wait for the process to complete before upgrading BillingCenter.
For more information about the Commission Payable Calculations process, see “Commission Payable Calculations Batch Processing” in the System Administration Guide.
